# Ideas

Pre-spec staging area for ideas I want to capture but not act on yet. When an
idea graduates, it becomes a real Spec Kit feature under `specs/NNN-slug/`.

## Conventions
- One file per idea: `YYYY-MM-DD-slug.md`
- Each file has a `Status:` line — one of: `brainstorm`, `shelved`, `promoted-to-spec`
- Capture *why not now* and *promotion criteria* so future-me knows what unblocks it
- Commit directly to `master` via PR (same flow as blog posts)
